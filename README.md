# Coffee Shop Sales Analysis ☕

This project provides data-driven insights to support the CEO in making informed business decisions. It analyzes sales trends over a 6-month period across three store locations:

- Lower Manhattan  
- Hell's Kitchen  
- Astoria  

## 🎯 Project Objective
To uncover sales patterns, customer behavior, and product performance in order to identify opportunities for growth and operational improvement.

## 💡 What I Did
- Inspected and explored raw data using Excel  
- Created a project mind map using Miro to plan the analysis approach  
- Built a timeline using Canva to manage project deadlines  
- Cleaned and transformed raw sales data using Databricks (SQL)  
- Analyzed customer purchasing patterns  
- Identified top-performing products and categories  
- Compared weekday vs weekend sales trends  
- Identified underperforming (low-selling) products  

## 🧰 Tools Used
- Databricks (SQL)  
- Excel  
- Power BI  
- Miro  
- Canva  

## 📈 Why This Project Matters
Understanding sales behavior enables businesses to optimize inventory, improve marketing strategies, and increase revenue.

## 📌 Key Takeaway
Data is more than numbers — it tells a story that can drive smarter, data-driven decisions.
